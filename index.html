About

Building the plumbing for agents that do real work.

I work at the intersection of large language models and the tools, memory, and guardrails that turn a chat interface into an agent capable of finishing a task end to end. My focus is the Anthropic ecosystem — Claude, MCP, and the patterns that make multi-step agent behaviour reliable rather than fragile.

This site is a running log of that work: shipped tools, experiments, and the occasional dead end worth documenting.

  • Multi-step agent orchestration & tool use
  • Claude / Anthropic API integrations
  • MCP servers & custom tool design
  • Evaluation & reliability for agentic systems
