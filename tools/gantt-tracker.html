Multi-Program Gantt & Critical Path Tracker

Paste tasks across one or more programs. Schedule dates, float, and the critical path are computed deterministically with the Critical Path Method (CPM) — no AI involved in the math. See each program on its own, or switch to the combined view to compare all programs side by side, color-coded, with slack (float) and resource assignments visible at a glance — plus an optional AI layer to narrate which programs carry the most schedule risk.

Simple (one program, no resource): id, name, duration, dependencies — e.g. B, Design, 8, A.
Multi-program: program, id, name, duration, dependencies — e.g. Data Migration, G, Extract, 4,.
With a resource assigned: add it as a 6th field — program, id, name, duration, dependencies, resource — e.g. Program C, C2, Design, 9, C1, Ravi.
Dependencies reference the id column, separated by semicolons. Leave a field blank (just the comma) if it doesn't apply.

AI Assist (optional)

Get a short narrative on which program(s) carry the most schedule risk, based on critical path length and near-critical float. Your Groq API key stays in your browser for this session only.

Part of the AexForge agentic AI toolkit — deterministic core, optional AI layer.
